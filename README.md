# UNC PhD Dissertation LaTeX Template

This repository is a LaTeX template for preparing a UNC PhD dissertation. The main document is `UNC_dissertation.tex`, and the official formatting reference included with the repository is `ETDGuide2013.pdf`.

Use this template as a starting point, but check the finished PDF against `ETDGuide2013.pdf` before submission. If your graduate program or the Graduate School gives newer instructions, follow the newer instructions.

## Repository contents

| File | Purpose |
| --- | --- |
| `UNC_dissertation.tex` | Main dissertation file. Compile this file to build the PDF. |
| `uncdissertation.cls` | UNC dissertation document class. Usually do not edit this file. |
| `uncdissertation.sty` | UNC dissertation style file. Edit the department name here before compiling. |
| `ch1.tex`, `ch2.tex`, `ch3.tex` | Chapter files loaded by `UNC_dissertation.tex`. Replace the sample text with your dissertation chapters. |
| `appA_Chapter1.tex`, `appA_Chapter2.tex`, `appA_Chapter3.tex` | Appendix files loaded after the chapters. Edit, remove, or add appendix files as needed. |
| `Bibliography.bib` | BibTeX database for references. Replace the sample references with your own sources. |
| `ETDGuide2013.pdf` | Official UNC ETD formatting instructions included with this repository. Use it as the formatting checklist. |

## Requirements

Install a LaTeX distribution that includes `pdflatex`, `bibtex`, and the common LaTeX packages used by the template.

Recommended options:

- macOS: MacTeX
- Windows: MiKTeX or TeX Live
- Linux: TeX Live
- Overleaf: upload the full repository and set `UNC_dissertation.tex` as the main file

`latexmk` is optional but recommended because it automatically runs LaTeX and BibTeX enough times to resolve citations, cross-references, the table of contents, and lists of figures/tables.

## First-time setup

1. Clone or download this repository.
2. Open `ETDGuide2013.pdf` and review the official formatting requirements.
3. Open `uncdissertation.sty` and change the department name.

   Find this line near the top of `uncdissertation.sty`:

   ```tex
   \def\mydepartment{Department of Economics}
   ```

   Replace it with your own department, for example:

   ```tex
   \def\mydepartment{Department of Biology}
   ```

   This value appears on the title page in the phrase “for the degree of Doctor of Philosophy in the ...”.

4. Open `UNC_dissertation.tex` and replace the title-page metadata:

   ```tex
   \thesistitle{THREE ESSAYS ON ...}
   \thesisauthor{Author}
   \thesisadvisor{Committee chair}
   \thesisreadera{Committee member 1}
   \thesisreaderb{Committee member 2}
   \thesismembera{Committee member 3}
   \thesismemberb{Committee member 4}
   ```

5. Replace the sample abstract and acknowledgments in `UNC_dissertation.tex`.
6. Replace the sample chapter and appendix text in the separate `.tex` files.
7. Replace the sample references in `Bibliography.bib` with your own bibliography entries.

## Building the dissertation PDF

From the repository root, run:

```bash
latexmk -pdf UNC_dissertation.tex
```

The output file will be:

```text
UNC_dissertation.pdf
```

If you do not have `latexmk`, run the commands manually:

```bash
pdflatex UNC_dissertation.tex
bibtex UNC_dissertation
pdflatex UNC_dissertation.tex
pdflatex UNC_dissertation.tex
```

Run the final `pdflatex` command more than once whenever you change citations, references, tables, figures, chapter titles, or page numbering.

To remove temporary LaTeX build files when using `latexmk`, run:

```bash
latexmk -c
```

## Editing the dissertation

### Front matter

The front matter is controlled mostly in `UNC_dissertation.tex`:

- title page metadata: `\thesistitle`, `\thesisauthor`, `\thesisadvisor`, and committee-member commands
- copyright page: `\thesismakecopyright`
- abstract: `\begin{abstract} ... \end{abstract}`
- acknowledgments: `\begin{acknowledgments} ... \end{acknowledgments}`
- table of contents, list of tables, and list of figures

Check `ETDGuide2013.pdf` for the required front-matter order and formatting.

### Chapters

The sample dissertation has three chapter blocks:

```tex
\chapter{TITLE OF CHAPTER 1}
\input{ch1}

\chapter{TITLE OF CHAPTER 2}
\input{ch2}

\chapter{TITLE OF CHAPTER 3 IS GENERATED BY AI SO IT IS EXTRAORDINARILY LONG}
\input{ch3}
```

To add a chapter, create a new file such as `ch4.tex`, then add another chapter block in `UNC_dissertation.tex`:

```tex
\chapter{TITLE OF CHAPTER 4}
\input{ch4}
```

To remove a chapter, delete or comment out its `\chapter{...}` and `\input{...}` lines.

### Appendices

Appendices are defined after `\appendix` in `UNC_dissertation.tex`. The template currently includes one appendix per chapter. Edit the existing appendix files, or add/remove appendix blocks to match your dissertation.

When adding an appendix, follow the existing pattern:

```tex
\begin{singlespace}
\refstepcounter{chapter}
\phantomsection
\chapter*{APPENDIX \thechapter: Appendix Title}
\addcontentsline{toc}{chapter}{APPENDIX \thechapter: Appendix Title}
\end{singlespace}
\input{appA_NewAppendix}
```

### References

References are managed with BibTeX. Add your references to `Bibliography.bib`, cite them in the text, and keep this block near the end of `UNC_dissertation.tex`:

```tex
\bibliographystyle{chicago}
\bibliography{Bibliography}
```

After adding or changing citations, run the full build sequence again so BibTeX and LaTeX can update the references correctly.

### Figures and tables

Use standard LaTeX figure and table environments. The template includes a sample landscape table in `ch1.tex`; use it as a model if you need a wide table.

After adding figures or tables, rebuild more than once so the list of figures, list of tables, captions, and cross-references are updated.

## Final formatting checklist

Before submitting, build a clean PDF and compare it against `ETDGuide2013.pdf`. At minimum, verify:

- title page contains the correct title, author, committee, year, and department
- `\mydepartment` in `uncdissertation.sty` has been changed from the sample department to your own department
- abstract, acknowledgments, table of contents, list of tables, and list of figures appear in the required order
- page numbering is correct in the front matter and main matter
- margins, spacing, title capitalization, and chapter headings satisfy the ETD guide
- all chapter titles and appendix titles are correct in the document and table of contents
- all figures and tables have correct captions and numbering
- all citations resolve and all references appear in the bibliography
- no placeholder text, sample author names, sample committee names, or sample references remain unless intentionally kept

## Troubleshooting

### Citations show as question marks

Run BibTeX and then run LaTeX twice:

```bash
pdflatex UNC_dissertation.tex
bibtex UNC_dissertation
pdflatex UNC_dissertation.tex
pdflatex UNC_dissertation.tex
```

### Table of contents, list of figures, or list of tables is wrong

Run LaTeX again. These lists are generated from auxiliary files and often need more than one compile pass.

### The department name is wrong on the title page

Edit `uncdissertation.sty` and change:

```tex
\def\mydepartment{Department of Economics}
```

to your own department. Then rebuild the PDF.



## Notes

- Keep `UNC_dissertation.tex` as the main file.
- Avoid editing `uncdissertation.cls` unless you intentionally need to change the document class.
- Edit `uncdissertation.sty` only for template-level settings such as the department name or global formatting behavior.
- Treat `ETDGuide2013.pdf` as the repository's official formatting reference, and verify the final PDF before filing.
